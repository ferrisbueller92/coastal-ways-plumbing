Water Compliance Cert Tool

Issue a branded, statutory-complete Water Efficiency Compliance Certificate for any rental property job. ~30 seconds per cert from job-finish to PDF-in-landlord's-inbox.

1
Copy the ChatGPT prompt below
One-click copy. Paste it into a new ChatGPT chat (any model — GPT-5, GPT-5-mini, GPT-4o all fine). The prompt has all of Coastal Ways' static info baked in.
2
Answer ChatGPT's 6 questions about the job
It'll ask you, one at a time: property address · inspection date · owner/agent · fixtures inspected · notable items · cert number. Plain language is fine — ChatGPT cleans it up.
3
Copy the values block ChatGPT gives back
It'll output a single code block with 9 fields. Each maps to a {{placeholder}} in the certificate template.
4
Open the certificate template + paste the values
Open template.html in your browser, use Cmd+F → Replace (or open the file in any text editor) to swap each {{...}} for the value ChatGPT gave you. Save.
5
Print to PDF → send to the landlord/agent
In your browser: File → Print → "Save as PDF" → A4 portrait. Email or AirDrop the PDF. Done.

What's covered (vs the typical plumber-issued cert)

This template uses the exact statutory wording from clause 10 of the Residential Tenancies Regulation 2019 (NSW). Most plumber-issued certs miss several of these items — leaving the landlord exposed if a tenant later disputes water usage charges at NCAT.
