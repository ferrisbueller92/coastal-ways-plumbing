Coastal Ways Plumbing — Email Signatures

Two ready-to-use signatures for Gmail and Outlook. Same source works in both. Copy the rendered block (not the raw code) for the easiest paste.

How to install (Gmail)

  1. Below, find your signature (Beau or Emma). Click and drag to select the entire signature box — start above the logo, drag down past the website link.
  2. Press ⌘ + C (Mac) or Ctrl + C (Windows) to copy.
  3. Open Gmail → click the gear icon top-right → See all settings.
  4. Scroll down to Signature → click Create new → name it "Coastal Ways".
  5. Click in the empty signature box → press ⌘ + V (or Ctrl + V) to paste.
  6. Under Signature defaults, set "Coastal Ways" for both NEW EMAILS and ON REPLY/FORWARD.
  7. Scroll to bottom → Save Changes. Send a test email to yourself to confirm it looks right.

How to install (Outlook — desktop app)

  1. Select and copy the rendered signature below (same as Gmail step 1–2).
  2. Open Outlook → File → Options → Mail → click Signatures…
  3. Click New → name it "Coastal Ways" → OK.
  4. Click into the editor box at the bottom → press ⌘ + V (or Ctrl + V) to paste.
  5. On the right under Choose default signature: set "Coastal Ways" for both New messages and Replies/forwards.
  6. Click OK. Open a new email to verify it appears.

How to install (Outlook on the web — outlook.com / Microsoft 365 web)

  1. Select and copy the rendered signature below.
  2. Top-right gear icon → View all Outlook settings → Mail → Compose and reply.
  3. Click into the signature box → paste with ⌘ + V (or Ctrl + V).
  4. Tick Automatically include my signature on new messages and on messages I forward or reply to.
  5. Click Save.
If the logo doesn't show: some clients block remote images by default. The recipient just needs to click "Display images" once — after that it's remembered. The signature still reads cleanly without the logo.
