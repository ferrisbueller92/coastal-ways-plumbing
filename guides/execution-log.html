How to add an item (when picking up next session)

  1. Edit guides/execution-log.html in the CWP repo
  2. Insert a new row in the relevant section (Today / 1 May queue / Beau-Emma queue / Sprint 3)
  3. Use the next sequence number
  4. Commit: cos: ledger - [what changed]
  5. Push — URL stays the same: coastalwaysplumbing.com/guides/execution-log.html

When an item completes

  1. Change the status pill from Todo or In Progress to Done
  2. Add a date in the link column if not already there
  3. Move it from a queue section to the build log section if it was previously queued
  4. Commit + push
